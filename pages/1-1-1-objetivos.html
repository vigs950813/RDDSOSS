Objetivos

El sistema operativo es el único de los programas que se encarga de interactuar con el hardware de la computadora las funciones principales así como su objetivo son:

  • Abstracción:

    Los programas no deben de presentar detalles de acceso al hardware, configuración de una computadora. ya que el sistema operativo se encarga de proporcionar una serie de abstracciones para que los programadores puedan resolver las necesidades de cada usuario, esto por ejemplo que la información esté organizada en sus archivos y directorios como dispositivos de almacenamiento.

  • Administración de recursos:

    Se refiere a la memoria, espacio de almacenamiento, tiempo de procesamiento, acceso a los controladores de sus dispositivos, etc, por ello los diferentes procesos que se lleve a cabo y se ejecuten debe de existir un orden para llevarlos a cabo, por ende el sistema operativo gestiona toda la asignación de recursos en donde puede implementar políticas para que se asignen en forma efectiva y dependiendo de las necesidades los recursos del sistema. sus archivos y directorios como dispositivos de almacenamiento.

  • Aislamiento:

    En un sistema multiusuario y multitarea cada proceso que genera el usuario no debe de preocupar que al mismo tiempo otro esté requiriendo recursos ya que el sistema implementa mecanismos en donde las funciones se ejecuten correctamente otorgando los recursos necesarios a cada uno esto dependerá de la importancia del proceso que cada uno esté ejecutando para el sistema operativo.

Catacterísticas

Las características principales del sistema operativo son:

  • Facilidad de uso;

    El sistema operativo se encarga de facilitar el uso de la computadora.

  • Eficiencia:

    Un sistema operativo es aquel que permite que los recursos de una computadora se utilizar de una manera eficiente

  • Capacidad para evolucionar:

    Un sistema operativo se construye de tal forma que se pueden desarrollar e implementar nuevas funciones de operatividad del mismo sin interferir con sus procesos y servicios que tiene.

Por lo que el objetivo del sistema operativo es administrar, controlar y distribuir los recursos requeridos tanto por el usuario como por el sistema en una forma organizada y dependiendo de la prioridad del mismo.
