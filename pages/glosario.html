Glosario

  • Argumento

    Un argumento de una función es un valor que se pasa a la función para que la utilice en su procesamiento.

  • CPU

    La CPU (Unidad Central de Procesamiento) es el componente de una computadora, encargada de ejecutar instrucciones de programas, realizar cálculos y coordinar las operaciones de otros componentes para procesar datos y llevar a cabo tareas.

  • Declaración

    línea de código que da instrucciones a la computadora para llevar a cabo una acción o definir una entidad, como variables, funciones o estructuras, en un programa.

  • Función

    Bloque de código que realiza una tarea específica y puede ser reutilizado llamándolo desde otros lugares de otro programa que puede ser de gran escala.

  • Hardware

    Se refiere a todos los componentes físicos y electrónicos de una computadora u otro dispositivo, como la CPU, memoria, disco duro, pantalla y dispositivos de entrada/salida, que trabajan juntos para ejecutar programas y realizar tareas.

  • Interrupción

    Señal que detiene momentáneamente la ejecución de un programa para manejar eventos urgentes o solicitudes del hardware, permitiendo una respuesta rápida y eficiente a situaciones como errores, solicitudes de entrada/salida y otros eventos importantes.

  • Proceso

    Es una instancia en ejecución de un programa, con su propio espacio de memoria y recursos asignados, que realiza tareas específicas de manera independiente dentro del sistema computacional.

  • RAM

    Del inglés, Random Access Memory (Memoria de Acceso Aleatorio), es el componente físico de una computadora que permite el almacenamiento temporal de información para su procesamiento y uso.

  • Script

    Conjunto de instrucciones cortas escritas en un lenguaje de programación para ejecutar tareas específicas.

  • Sistema operativo

    Es el software que administra recursos y facilita la interacción entre el hardware y otros programas en una computadora.

  • Software

    Es el conjunto de programas, instrucciones y datos que controlan el funcionamiento de una computadora o dispositivo, permitiendo realizar tareas y procesar información de manera programada.
