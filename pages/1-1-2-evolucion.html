Bienvenida

Estimado alumno y alumna de la unidad de aprendizaje de sistemas operativos, perteneciente al cuarto semestre del programa académico de ingeniería en sistemas computacionales que se imparte en diferentes campus del Instituto Politécnico Nacional, te damos la más cálida de las bienvenida a este recurso didáctico digital (RDD) de la unidad temática: 1 estructura del sistema operativo, el cual se construyó con la intención de apoyarte en tu proceso de aprendizaje. Nuestro propósito es que este RDD te facilite la construcción de tus aprendizajes y que disfrutes de esta experiencia didáctica.

Metodología

Dentro de esta unidad temática: "Estructura de un Sistema Operativo", fortalecerás el aprendizaje autónomo, el trabajo individual y colaborativo. En este sentido, es de importancia resaltar que por cada 4.5 horas de clases por semana, requeriras de 1.5 horas de trabajo individual y autónomo.
1.1 Aspectos del sistema Operativo

Con el estudio de este tema comprenderás la importancia que tiene el sistema operativo con relación al funcionamiento de los elementos que conforman a los equipos de cómputo, así como sus antecedentes históricos.

1.2 Fundamentos del núcleo del sistema operativo

Al estudiar este tema aprenderás la importancia del kernel o núcleo de un sistema operativo y el papel que juega como la interfaz entre el software y el hardware, es por ello que identificarás porque el kernel es conocido como el corazón de un sistema operativo.

1.3 Modos de operación de un sistema operativo

Al trabajar con este tema tendrás los elementos para comprender el funcionamiento de los modos de operación de un sistema operativo siendo estos una guía que te ayude a visualizar, identificar y asociar los modos de operación de un sistema operativo con el hardware y software de un computadora.

1.4 Llamadas a sistema

Al abordar este tema podrás construir e identificar la importancia de las llamadas del sistema en los procesos, y cómo estos se utilizan para entablar comunicación con el kernel, directorios, archivos y seguridad en cual es parte medular para el sistema operativo

1.5 Shell

Este tema te ayudará a comprender la funcionalidad que tiene el shell como el mediador entre el usuario y el núcleo del sistema operativo, así mismo identificarás argumentos, líneas de comando, estructuras de control cadenas y funciones que son parte medular para que los procesos que generé el sistema operativo se lleven a cabo.
